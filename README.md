# Xor-Shift
Семейство Xor-Shift генераторов 

Используемая библиотека для больших чисел [LibTomMath](https://www.libtom.net/LibTomMath/)

 ## Теоретическое описание
 
 Xorshift - семейство генераторов регистров сдвига, представляют собой класс генераторов псевдослучайных чисел , открытых Джорджем Марсалья. Подробное описание алгоритмов и рекомендуемых к использованию констант приведенов в [статье](https://www.jstatsoft.org/article/view/v008i14) Дж. Марсалья.
